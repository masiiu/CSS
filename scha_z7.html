    2. Styl Lokalny pozwala na formatowanie konkretnego elementu.

  2. Składnia teoretyczna ( z użyciem ramki otaczającej może być tabela o jednej komórce, zastosuj „znaki specjalne” do napisania definicji stylu lokalnego tak aby instrukcja CSS nie wykonywała się a tylko wyświetlała wewnątrz ramki [  spacja niełamana] przykład „znaku specjalnego”)
  3. Co może być selektorem

    4. Dowolny znacznik

  4. Co wpisujemy jako "cechę". Jakie jest znaczenie średników

    5. Jako "cecha" czy inaczej własność bądź właściwość (ang. "property") należy wpisać o jakie konkretnie atrybuty formatowania nam chodzi(np:kolor tekstu -color). Jednemu selktorowi możemy nadać kilka atrybutów(cech). Są one wtedy rozdzielone średnikami.

  5. Co wpisujemy jako "wartość" (w deklaracji stylu).

    6. Jako wyraz "wartość" (w deklaracji stylu) wpisujemy dokładną wartość atrybutu (np:dla koloru tekstu będzie to: red,green). Wartość zależy ściśle od cechy, po której stoi w deklaracji.
